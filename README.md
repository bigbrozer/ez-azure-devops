## Development

### Requirements

* [asdf](https://asdf-vm.com/)

### Install tools

Install [Task](https://taskfile.dev/):

```sh
asdf plugin add task
asdf install
```

### Virtual environment

Init your python environment with:

```bash
task venv
```

You're all set !

### Tests

Run all tests with:

```bash
task tests
```
